# mocli

Command line interface for Mobingi API.

### Build

This tool uses [`dep`](https://github.com/golang/dep) for dependency management. You can install `dep` via

```
$ go get -u github.com/golang/dep/cmd/dep
```

To build the tool, run

```
$ go build -v
```
